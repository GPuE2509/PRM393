# prm393 - Personal Expense Manager

Flutter app for personal expense management with local SQLite storage.

## Current Scope

- Authentication UI: login and register
- Expense dashboard: view total spending and expense list
- Expense operations: add and delete expenses
- Local persistence: users, expenses, and session data

## Tech Stack

- Flutter
- sqflite
- shared_preferences

## Run

1. flutter pub get
2. flutter run
