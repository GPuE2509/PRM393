# Gallery Banner Images

## Instructions

Place your gallery banner image here as `banner.png`.

### Recommended Specifications:
- **Format**: PNG (transparent background recommended)
- **Size**: 1200x300 pixels (4:1 ratio) or similar
- **Purpose**: Gallery header/logo for Home Screen

### How to add your banner:
1. Save your banner image as `banner.png` in this directory
2. The app will automatically load it from `assets/images/banner.png`

### Alternative: Using a placeholder
If you don't have a banner yet, the app will display a styled gradient placeholder with the gallery logo.
